+ Bool 대수 식

    - 분배 / 결합 / 항등 / 교환 법칙
    - 논리합(OR), 논리곱(AND)
    - 1 * 1 = 1
    - 1 + 0 = 1 
    - (a')' = a
    - a + 1 = 1 // 일반적인 연산과 같음
    - (a + b)' = a' * b' / (a * b)' = a' + b' // 드모르간법칙
    - a + (b * c) = (a + b) * (a + c) // 분배 2법칙
