# Insertion Sort

```
 1. 앞에서부터 차례로 정렬한 자료배열과 비교하여 적절한 위치를 찾아 삽입하는 것이 삽입정렬입니다.

 2. 바깥의 큰 루프를 n번 시행 할 때마다 n+1 의 크기를 가지는 정렬된 배열을 가지도록 설계했고
    
    정렬된 배열의 다음 인덱스의 값을 key로 설정하여 정렬된 배열과 비교하여 insertion 하였습니다.

 3. 따라서 조건에 맞는 위치에 값을 삽입하며 정렬되어야할 배열의 부분집합의 크기를 하나씩 늘려가면서

    정렬을 수행하는 알고리즘을 고안했습니다.
```

# Bubble Sort

```
 1. 서로 인접한 두 원소를 비교하여 조건에 맞게 인접한 값들을 교환하며 정렬하는 알고리즘이 버블정렬입니다.

 2-1. 이미 정렬된 값들 중에 key의 위치를 찾아 삽입하는 insertion sort 와 달리 정렬되지 않은 값들을 매번 모두 탐색하기 때문에

    정렬을 시행하지 않은 값들이 이미 정렬이 돼있을 경우를 확인해 루프를 종료할 수 있도록 했습니다.

 2-2. 배열의 크기 - 1 번만큼 반복하며 앞의 원소와의 대소를 비교해 조건에 맞게 값을 서로 교환하고

    swap()이라는 함수를 선언해 bit swap 으로 값을 교환하게 하였습니다.

 3. 따라서 0번 인덱스의 원소부터 시작해 정렬이 완료 될 때까지 서로 인접한 원소들과의 대소관계를 비교하여 교환하는

    알고리즘을 구현했습니다.(결과적으론 각 과정마다 뒤에 있는 원소부터 정렬이 완료됨)
    
```
